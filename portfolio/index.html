About Me

Starting out at the age of 17, I never stopped learning new programming skills and languages. Early on, I started creating websites out of curiosity and for fun. I became proficient in web design using HTML, CSS, and jQuery.

Download My Resume

Beyond front-end development, I explored backend technologies and other non-web-only technologies. This passion led to my decision to work as a freelance web developer. The success and enjoyment I find in this profession keep my passion burning alive.

Starting with frontend development using modern frameworks like ReactJS, I've also become increasingly proficient in backend development with Node.js and Laravel frameworks. I love both worlds nowadays!

Not only am I skilled in development, but I'm also adept at deploying web applications on Linux servers. I have a strong understanding of CI/CD pipelines with GitHub Actions and GitLab Runner.

Besides development and deployment, I've explored robotics as a hobby. When I feel like taking a break from coding, I enjoy working with my Arduino learning kit. I primarily work on SAAS microservice-based projects and contribute to open-source projects.
